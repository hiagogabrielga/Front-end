Introdução a HTML e CSS

HTML (HyperText Markup Language) e CSS (Cascading Style Sheets) são fundamentais para a construção e estilização de páginas web. HTML define a estrutura e o conteúdo de uma página, enquanto o CSS é usado para controlar a apresentação visual desse conteúdo. Juntas, essas tecnologias permitem criar páginas web atraentes e funcionais.

HTML (HyperText Markup Language):

HTML é a linguagem de marcação usada para estruturar o conteúdo da web. Ela define os diferentes elementos de uma página, como cabeçalhos, parágrafos, listas, links e imagens. Cada elemento HTML é representado por uma tag, e a estrutura básica de um documento HTML inclui a declaração <!DOCTYPE html>, a tag <html>, e as seções <head> e <body>.

CSS (Cascading Style Sheets):

CSS é uma linguagem de estilo que controla a aparência dos elementos HTML. Com CSS, você pode definir cores, fontes, espaçamentos, layouts e outros aspectos visuais das páginas web. O CSS pode ser incluído diretamente no HTML, em um arquivo separado ou em um bloco de estilo dentro do documento HTML.

Exemplos de Sites que Utilizam HTML e CSS:

Apple: Usa HTML e CSS para criar um design limpo e moderno em seu site, com foco na apresentação visual dos produtos.
Amazon: Emprega HTML e CSS para estruturar e estilizar suas páginas de produtos e listas de compras.
The Guardian: Utiliza HTML e CSS para formatar suas notícias e artigos de forma clara e atraente.

Estrutura Básica:

HTML: A estrutura básica de um documento HTML é composta por várias tags. A tag <html> envolve todo o documento, a tag <head> contém metadados e links para folhas de estilo, e a tag <body> contém o conteúdo visível da página. Dentro do <body>, você pode ter tags como <header>, <nav>, <main>, <section>, <article>, <footer>, e outras para organizar o conteúdo.

CSS: A estrutura básica do CSS é composta por seletores e regras de estilo. Um seletor identifica o elemento HTML a ser estilizado, e uma regra de estilo define as propriedades e valores a serem aplicados. O CSS pode ser incluído em um documento HTML dentro da tag <style>, ou referenciado através de um link para um arquivo externo com a tag < link>.
